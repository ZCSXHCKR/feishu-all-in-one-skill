# 飞书全场景办公自动化All-in-One Skill
> OpenClaw生态首个整合飞书所有常用功能的一站式Skill，开箱即用，无需安装多个零散Skill。

## ✨ 核心优势
### 🎯 功能全覆盖
- 文档/知识库：读写、批量导出、权限管理
- 消息：发送、接收、群管理、@回复、表情互动
- 日历/会议：日程管理、自动排期、妙记转录、会议总结
- 多维表格：CRUD、批量操作、数据同步
- 云盘：文件上传下载、批量管理
- 审批：自动提交、状态查询、通知提醒
- 考勤：自动统计、迟到早退预警
- 任务：待办同步、提醒、状态更新

### 🚀 开箱即用
- 统一配置：只需配置一次飞书应用凭证，所有功能自动生效
- 零学习成本：自然语言指令即可调用所有功能
- 兼容所有飞书版本：国内/国际版、企业版/个人版全支持
- 低资源占用：比零散安装10个Skill节省70%内存和Token消耗

### 💰 商业化方案
- **免费版**：基础功能（文档读写、消息发送、日历查询），限100次/月调用
- **专业版**：149元/年，全功能开放，1000次/月调用，技术支持
- **企业版**：1499元/年，无限次调用，专属客服，定制化功能开发
- **私有化部署**：14999元/年，企业私有部署，数据100%留存在企业内部
