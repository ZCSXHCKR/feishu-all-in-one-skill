## 📦 License Key 交付方式

### 方式1：自动发放（推荐）

**技术实现：**
1. 用户付款后，系统自动识别付款信息
2. 根据付款金额判断版本类型
3. 自动调用 `scripts/license.js` 生成 License Key
4. 通过飞书机器人自动发送给用户

**交付时间：** 1-5分钟

### 方式2：人工发放（当前方案）

**技术实现：**
1. 用户付款后联系开发者（微信/邮件）
2. 开发者手动调用 `scripts/license.js` 生成 License Key
3. 通过飞书/微信/邮件发送给用户

**交付时间：** 10-30分钟（人工响应时间）

### 方式3：邮件自动发送（未来方案）

**技术实现：**
1. 用户填写邮箱地址
2. 系统自动生成 License Key
3. 发送激活邮件（含 License Key + 激活指南）
4. 用户收到邮件后立即激活

**所需资源：**
- ⏳ 邮件发送服务（如阿里云邮件推送）
- ⏳ Web 表单系统

## 🔐 License Key 生成流程

### 使用 License 管理脚本

```bash
# 生成专业版 License（1年有效期）
node scripts/license.js generate pro 365

# 输出示例：
✅ License Key 生成成功！
  版本：专业版
  价格：¥149/年
  有效期：365天
  到期日：2027-04-05
  
  🔑 您的License Key：
  pro-1743849600000-abc123xyz
  
  ⚠️ 请妥善保管此Key，不要泄露给他人！
```

### License Key 格式说明

```
{edition}-{timestamp}-{hash}
```

- **edition**: 版本类型（free/pro/enterprise/private）
- **timestamp**: 过期时间戳（毫秒）
- **hash**: 随机生成的哈希值（唯一标识）

| 服务等级 | 响应时间 | 解决时间 | 适用版本 |

| **社区支持** | 不保证 | 不保证 | 免费版 |
| **标准支持** | 48h | 72h | 专业版 |
| **优先支持** | 24h | 48h | 企业版 |

### 常见问题解答（FAQ）

**Q: License Key 立即生效吗？**
A: 是的，填入 License Key 后立即生效，无需重启。

**Q: 可以升级版本吗？**
A: 可以，补差价即可升级，立即生效。

**Q: 可以退款吗？**
A: 支持7天无理由退款，30天内功能不符退70%。

**Q: License Key 可以多人共用吗？**
A: 不行，每个 License Key 只能激活一次。企业版可添加5个子账号。

**Q: License 过期后怎么办？**
A: 过期后自动降级为免费版，续费后恢复全部功能。
